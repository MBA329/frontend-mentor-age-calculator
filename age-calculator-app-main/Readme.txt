## Age Calculator

A responsive web application that accurately calculates your age in years, months, and days based on your date of birth. This lightweight tool is designed for quick and easy use on any device, whether it's a desktop, tablet, or mobile phone.

### Features:
- **Responsive Design**: Optimized for all screen sizes, ensuring a smooth experience across devices.
- **Accurate Calculation**: Accounts for varying month lengths and leap years.
- **User-Friendly Interface**: Simple and intuitive design for easy input and clear results.

### Tech Stack:
- **HTML/CSS**: For structure and responsive design.
- **JavaScript**: For calculating and displaying age.

### Getting Started:
1. Clone the repository: `git clone https://github.com/yourusername/age-calculator.git`
2. Open `index.html` in your browser to start using the app.
